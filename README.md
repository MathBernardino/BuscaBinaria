# BuscaBinaria

<b>Explicação Sobre o Site</b><br>
Um caça palavras que na parte superior irá ter as perguntas com uma caixa para colocar a resposta, junto com um botão para dar uma dica se tiver dificuldade na resposta. Após colocar a resposta o site irá informar se está certo ou errado, após isso terá um caça palavra para que o usuário possa achar a sua resposta.
O caça palavras será formado por botões em que quando clicados irão ficar com uma cor para identificar a palavra
